# KeshoSave

A fictional Kenyan personal finance web application built as a lab assignment for the Web Application Development course at Strathmore University.

**Live site:** [munene-mutuku.github.io/web-application-y1s3-project](https://munene-mutuku.github.io/web-application-y1s3-project/)

---

## About

KeshoSave is a fintech concept app targeting Kenyan youth, students, and young professionals. It offers budgeting tools, savings goals, expense tracking, and M-Pesa integration — all in one free platform.

**Tagline:** Save smarter, live better.
